# Krallıkların Terazisi

Türkçe strateji tabanlı kart oyunu. Krallığını yönet, ittifaklar kur, düşmanlarına saldır!

## 🎮 Oyunu Oyna

[Oyunu Buradan Oynayın](https://[kullanıcı-adınız].github.io/anGra/)

## 🎯 Özellikler

- 🏰 **Krallık Yönetimi**: Binalar inşa et, askerleri yönet
- ⚔️ **Savaş Sistemi**: Zar atarak düşmanlarına saldır
- 🤝 **İttifaklar**: Diplomatik ilişkiler kur
- 🎴 **Kart Sistemi**: Strateji kartları kullan
- 🤖 **Yapay Zeka**: Bot oyuncularla oyna
- 📊 **Teknoloji Ağacı**: Krallığını geliştir

## 🚀 Yerel Kurulum

```bash
# Depoyu klonla
git clone https://github.com/[kullanıcı-adınız]/anGra.git
cd anGra

# Basit HTTP sunucusu başlat
python -m http.server 8000

# Tarayıcıda aç
http://localhost:8000
```

## 📖 Nasıl Oynanır

1. **Oyun Başlangıcı**: 2-4 oyuncu seç, bot sayısını belirle
2. **Tur Sistemi**: Her oyuncu 2 aksiyon harcayabilir
3. **Kartlar**: Pazar'dan kart satın al, krallığına yerleştir
4. **Saldırı**: Düşman binalarına saldır, zar at
5. **Diplomasi**: İttifak kur veya boz
6. **Zafer**: Saray'ı yık, vassal yap veya en yüksek DP'ye ulaş

## 🎴 Kart Tipleri

- **🏰 Binalar**: Çiftlik, Kışla, Duvar, Pazar
- **⚔️ Askerler**: Piyade, Okçu, Süvari
- **🎭 Diplomasi**: Casusluk, Propaganda, Suikast, Nifak Tohumu
- **🔬 Teknoloji**: Yemek, Silah, Zırh kartları
